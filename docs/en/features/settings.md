# App Settings

PureSend offers rich customization options to adjust the app's appearance and behavior.

## Appearance Settings

### Theme

- **Light Mode**: Bright interface theme
- **Dark Mode**: Dark interface theme
- **Follow System**: Automatically match the operating system's theme

### Language

- **中文**: Simplified Chinese interface
- **English**: English interface
- **Follow System**: Automatically match the operating system's language

### Tab Bar Layout

- Customize the bottom navigation bar's tab display and order

### Font Size

- **Presets**: Small / Medium / Large / Extra Large
- **Custom**: Freely adjust font scaling ratio
- **Follow System**: Use system default font size

## Receive Settings

### Auto Receive

- When enabled, incoming transfer requests are automatically accepted without manual approval

### File Overwrite

- Configure how to handle files with the same name
- Options to overwrite or auto-rename

### Receive Directory

- Set the default save location for received files

## Advanced Settings

### Transfer Encryption

- Enable/disable AES-256-GCM transfer encryption
- When enabled, all transfer data is encrypted

### Dynamic Compression

- Enable/disable zstd dynamic compression
- Adjust compression mode and level

### Privacy Mode

- When enabled, transfer history is not recorded

### Developer Options

- Enable/disable WebView DevTools
