# Transfer History

PureSend automatically records detailed information for each transfer, making it easy to view and manage your transfer records.

## Overview

### Automatic Recording

After each transfer completes, the system automatically records:

- **Transfer Direction**: Send or receive
- **Transfer Status**: Completed, failed, cancelled, interrupted, etc.
- **Device Info**: Peer device name and IP
- **File List**: Transferred file names and sizes
- **Transfer Statistics**: Total size, duration, average speed

### Viewing History

- View all transfer records on the "History" page
- Records are sorted by time in reverse chronological order
- Expand individual records to view detailed file lists

## Filtering & Sorting

### Filter Options

- **By Direction**: All / Send only / Receive only
- **By Status**: All / Completed / Failed / Cancelled / Interrupted

### Sort Options

- Sort by transfer time
- Sort by file size

## Privacy Mode

- When privacy mode is enabled in settings, no new transfer history is recorded
- Existing history records are not automatically deleted
- All history can be manually cleared

## Management

- **Clear History**: One-click clear all transfer history
- History is stored locally and never uploaded to any server
