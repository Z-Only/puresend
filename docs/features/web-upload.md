# Web 上传

Web 上传功能允许其他设备通过浏览器向您的 PureSend 应用上传文件，无需在发送方安装应用。

## 使用方法

### 开启 Web 上传

1. 在「接收」页面，将接收模式切换为「Web 上传」
2. 系统将启动 HTTP 上传服务器
3. 生成的上传链接和二维码会显示在界面上

### 接收文件

- 将上传链接或二维码分享给发送方
- 发送方在浏览器中打开链接
- 发送方选择文件并上传

### 发送方（浏览器端）操作

1. 在浏览器中打开上传链接
2. 点击「选择文件」或将文件拖拽到上传区域
3. 确认文件列表后点击「上传」
4. 等待上传完成

## 访问控制

### IP 审批

- 新 IP 地址首次访问时，会向接收方发送访问请求
- 接收方可以选择「接受」或「拒绝」该 IP 的上传权限
- 已授权的 IP 可以直接上传，无需重复审批

## 传输特性

- **分块上传**: 大文件自动分块上传，支持断点续传
- **加密传输**: 支持 AES-256-GCM 加密，保护传输中的数据
- **进度显示**: 浏览器端和应用端均显示实时上传进度
- **文件覆盖**: 可在设置中配置同名文件的覆盖策略

## 注意事项

- 接收方和发送方需在同一局域网或可互相访问的网络中
- 接收期间需保持 PureSend 应用运行
- 上传的文件保存在设置的接收目录中
