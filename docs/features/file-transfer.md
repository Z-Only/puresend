# 文件传输

PureSend 提供快速、安全的局域网文件传输功能。

## 工作原理

PureSend 使用 WebRTC 技术实现点对点传输：

1. **发现设备**: 通过信令服务器发现同一网络中的其他设备
2. **建立连接**: 使用 WebRTC 建立直接的 P2P 连接
3. **传输数据**: 文件数据直接在设备间传输，不经过服务器

## 功能特性

### 发送文件

- **支持类型**: 任意类型的文件
- **批量发送**: 支持同时选择多个文件或整个文件夹
- **断点续传**: 支持传输中断后继续传输
- **传输速度**: 局域网内可达 100Mbps+

### 接收文件

- **自动发现**: 自动发现附近的发送设备
- **二维码扫描**: 扫描二维码快速连接
- **配对码输入**: 手动输入配对码连接
- **保存位置**: 自定义接收文件的保存位置

## 安全性

### 端到端加密

所有传输的数据都使用 DTLS-SRTP 进行加密：

- 数据在发送端加密
- 只有接收端能解密
- 服务器无法查看传输内容

### 隐私保护

- 文件不经过任何中转服务器
- 传输完成后不保留任何记录
- 支持设置传输密码

## 使用技巧

### 提高传输速度

1. 确保设备在同一局域网
2. 关闭 VPN 和代理
3. 使用 5GHz WiFi 网络

### 大文件传输

- 支持传输超大文件（GB 级别）
- 自动分块传输
- 实时显示传输进度

### 跨平台传输

支持任意平台之间的文件传输：

- macOS ↔ Windows
- Linux ↔ Android
- 桌面端 ↔ 移动端
