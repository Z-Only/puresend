# 云盘中转

PureSend 支持通过云盘服务进行文件中转传输，实现跨网络、跨设备的文件共享。

## 概述

云盘中转功能允许您将文件上传到云盘，然后在其他设备上下载。适用于以下场景：

- 设备不在同一局域网
- 需要跨网络传输大文件
- 需要长期保存分享文件
- 接收方不在线时的异步传输

## 支持的云盘类型

### WebDAV

支持标准 WebDAV 协议，兼容多种云存储服务：

- **坚果云** - 国内流行的云存储服务
- **NextCloud** - 开源私有云解决方案
- **Alist** - 支持多种网盘的列表程序
- 其他支持 WebDAV 协议的服务

### 阿里云 OSS

阿里云对象存储服务（Object Storage Service），适合：

- 大文件存储和传输
- 高频访问的文件分发
- 企业级存储需求

### 阿里云盘

阿里云盘个人版，适合：

- 个人用户日常使用
- 大容量存储需求
- 便捷的文件分享

## 功能特性

### 多账号管理

- 支持添加多个云盘账号
- 账号状态实时显示（已连接、未连接、无效）
- 快速切换不同账号

### 文件操作

- **浏览目录** - 查看云盘文件和目录结构
- **上传文件** - 将本地文件上传到云盘
- **下载文件** - 从云盘下载文件到本地
- **创建目录** - 在云盘上创建新目录

### 安全保护

- **凭证加密** - 使用 AES-256-GCM 加密存储密码、密钥等敏感信息
- **设备绑定** - 加密密钥基于设备标识派生，确保凭证仅在本机可用
- **安全传输** - 所有云盘通信均使用 HTTPS 加密

## 使用方法

### 添加云盘账号

1. 进入「设置」页面
2. 选择「云盘账号」选项
3. 点击「添加账号」
4. 选择云盘类型并填写凭证信息：
   - **WebDAV**: 服务器地址、用户名、密码
   - **阿里云 OSS**: Bucket、Region、AccessKey ID、AccessKey Secret
   - **阿里云盘**: Refresh Token
5. 点击「测试连接」验证凭证
6. 保存账号

### 上传文件到云盘

1. 进入「云盘」页面
2. 选择已配置的云盘账号
3. 在上传面板中选择本地文件
4. 选择或创建云盘目标目录
5. 开始上传

### 从云盘下载文件

1. 进入「云盘」页面
2. 选择已配置的云盘账号
3. 在下载面板中浏览云盘目录
4. 选择要下载的文件
5. 选择本地保存位置
6. 开始下载

## 凭证获取指南

### WebDAV（坚果云）

1. 登录坚果云网页版
2. 进入「账户信息」>「安全选项」
3. 添加应用密码
4. 使用生成的密码作为 WebDAV 密码

服务器地址格式：`https://dav.jianguoyun.com/dav/`

### 阿里云 OSS

1. 登录阿里云控制台
2. 进入对象存储 OSS
3. 创建或选择 Bucket
4. 在「访问控制」中创建 AccessKey

### 阿里云盘

1. 登录阿里云盘网页版
2. 打开浏览器开发者工具
3. 在网络请求中找到 `refresh_token` 参数
4. 或使用第三方工具获取

## 注意事项

- 云盘传输速度取决于网络带宽和云盘服务限制
- 部分云盘服务可能有流量或存储限制
- 建议在 Wi-Fi 环境下传输大文件
- 定期检查云盘存储空间
