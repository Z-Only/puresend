# 传输历史

PureSend 自动记录每次传输的详细信息，方便您查看和管理历史传输记录。

## 功能概览

### 自动记录

每次传输完成后，系统自动记录以下信息：

- **传输方向**: 发送或接收
- **传输状态**: 完成、失败、取消、中断等
- **设备信息**: 对端设备名称和 IP
- **文件列表**: 传输的文件名称和大小
- **传输统计**: 总大小、耗时、平均速度

### 查看历史

- 在「历史」页面查看所有传输记录
- 记录按时间倒序排列
- 展开单条记录可查看详细文件列表

## 筛选与排序

### 筛选条件

- **按方向**: 全部 / 仅发送 / 仅接收
- **按状态**: 全部 / 已完成 / 失败 / 取消 / 中断

### 排序方式

- 按传输时间排序
- 按文件大小排序

## 隐私模式

- 在设置中开启隐私模式后，系统将不再记录新的传输历史
- 已有的历史记录不会被自动清除
- 可手动清空全部历史记录

## 管理操作

- **清空历史**: 一键清空所有传输历史记录
- 历史记录保存在本地，不会上传到任何服务器
