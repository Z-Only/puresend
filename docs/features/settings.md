# 应用设置

PureSend 提供丰富的自定义选项，您可以根据需要调整应用的外观和行为。

## 外观设置

### 主题

- **浅色模式**: 明亮的界面主题
- **深色模式**: 暗色的界面主题
- **跟随系统**: 自动匹配操作系统的主题设置

### 语言

- **中文**: 简体中文界面
- **English**: 英文界面
- **跟随系统**: 自动匹配操作系统的语言设置

### Tab 栏布局

- 自定义底部导航栏的 Tab 显示和顺序

### 字体大小

- **预设大小**: 小 / 中 / 大 / 超大
- **自定义**: 自由调节字体缩放比例
- **跟随系统**: 使用系统默认字体大小

## 接收设置

### 自动接收

- 开启后，收到传输请求时自动接受，无需手动审批

### 文件覆盖

- 配置接收到同名文件时的处理策略
- 可选择覆盖或自动重命名

### 接收目录

- 设置接收文件的默认保存位置

## 高级设置

### 传输加密

- 开启/关闭 AES-256-GCM 传输加密
- 开启后所有传输数据均会加密

### 动态压缩

- 开启/关闭 zstd 动态压缩
- 调节压缩模式和级别

### 隐私模式

- 开启后不再记录传输历史

### 开发者选项

- 开启/关闭 WebView DevTools
